Upload Image\n\n```\n┌─────────────────────────────────────────────────────────────────┐\n│ 1. Admin selects image file                                      │\n└─────────────────────────────────────────────────────────────────┘\n                              ↓\n┌─────────────────────────────────────────────────────────────────┐\n│ 2. JavaScript: uploadImage(event)                                │\n│    - Create FormData with image and product_id                  │\n│    - Send POST to /product-images/upload                        │\n└─────────────────────────────────────────────────────────────────┘\n                              ↓\n┌─────────────────────────────────────────────────────────────────┐\n│ 3. ProductImageController::upload()                              │\n│    - Validate image (type, size)                                │\n│    - Generate unique filename                                   │\n│    - Move file to public/images/products/                       │\n└─────────────────────────────────────────────────────────────────┘\n                              ↓\n┌─────────────────────────────────────────────────────────────────┐\n│ 4. Create ProductImage record                                    │\n│    - product_id: from request                                   │\n│    - image: path/to/image.jpg                                   │\n│    - is_main: false                                             │\n│    - order: max(order) + 1                                      │\n└─────────────────────────────────────────────────────────────────┘\n                              ↓\n┌─────────────────────────────────────────────────────────────────┐\n│ 5. Return JSON response                                          │\n│    { success: true, message: \"Image uploaded successfully\" }    │\n└─────────────────────────────────────────────────────────────────┘\n                              ↓\n┌─────────────────────────────────────────────────────────────────┐\n│ 6. JavaScript: Reload page or update UI                          │\n│    - Show success message                                       │\n│    - Refresh image gallery                                      │\n└─────────────────────────────────────────────────────────────────┘\n```\n\n## Request Flow - Set Main Image\n\n```\n┌─────────────────────────────────────────────────────────────────┐\n│ 1.
